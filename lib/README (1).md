# Evaluacion Set y Map

 ## Metodo A: filterAndSortSupplies

 Implementacion usada: TreeSet
 Explicacion: El enunciado especificaba que el resultado debe quedar ordenado
 de manera ascendente y que elimine duplicados logicos para lo cual en este caso
 he elegido Set por que esta implementacion no permite duplicados.

## Metodo B: groupCodesByStock

Implementacion utilizada para el Map: TreeMap
Implementacion utilizada para los Sets internos: LinkedHashSet
Explicacion: Se utilizo el TreeMap porque ordena las claves alfabeticamente y 
el linked hash set porque almacena conforme vayan ingresando los datos.
